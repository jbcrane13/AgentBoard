# V2 UI Port (Toolbar + Columns + Chat Input) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port AgentBoard-v2's compact toolbar, narrower/darker board columns, and slim chat input chrome into the current AgentBoard `WorkScreen` and `ChatScreen` while preserving all functionality and the Neumorphic theme.

**Architecture:** Layout-only port. Modifies two existing screens (`WorkScreen.swift`, `ChatScreen.swift`) plus one component (`VoiceRecordingButton`). No new files, no new types, no store/model/service changes. Reuses existing `WorkStore.searchText` filtering and existing accessibility identifiers.

**Tech Stack:** SwiftUI (Swift 6, macOS 15+), Neumorphic theme (`NeuPalette`, `neuExtruded`, `neuRecessed`).

**Spec:** `docs/superpowers/specs/2026-04-29-v2-ui-port-design.md`

    private var filterRepositoryPicker: some View {
        Group {
            if appModel.settingsStore.repositories.count > 1 {
                Picker("Repo", selection: $selectedRepo) {
                    Text("All repos").tag("all")
                    ForEach(appModel.settingsStore.repositories) { repo in
                        Text(repo.shortName).tag(repo.fullName)
                    }
                }
                .pickerStyle(.menu)
                .tint(NeuPalette.accentOrange)
            } else {
                AgentBoardPill(text: "All repos", color: NeuPalette.accentOrange)

        }

            set: { appModel.workStore.searchText = $0 }
        )

                .fill(color)
                .frame(width: 7, height: 7)
                .shadow(color: color.opacity(0.6), radius: 4)
            Text("\(count)")
                .font(.system(.caption, design: .rounded, weight: .semibold))
                .monospacedDigit()
                .foregroundStyle(NeuPalette.textPrimary)
            Text(label)
                .font(.caption)
                .foregroundStyle(NeuPalette.textSecondary)
